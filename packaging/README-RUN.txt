Monitor Information 0.2.0

This is a compact framework-dependent Windows x64 build.

Required runtime:
.NET 10 Desktop Runtime for Windows x64

Official Microsoft download page:
https://dotnet.microsoft.com/en-us/download/dotnet/10.0

Install "Desktop Runtime" for Windows x64, then run:
MonitorInformation.exe

If the runtime is missing, MonitorInformation.exe uses the native .NET
apphost check before the application starts and shows Microsoft's runtime
installation link.
